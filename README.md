# 🎮 StatCraft

**StatCraft** is a gamified **productivity** and **fitness tracker** that turns your study sessions and workouts into an RPG style progression system. Earn XP based on time studied and calories burned, unlock achievements, and level up your personal stats.

---

## 🚀 Features

- 📚 **Study Session Tracking**  
  Log your study time and stay consistent  

- 💪 **Workout Tracking**  
  Record workouts along with **calories burned**  

- 🔥 **Calorie-Based Progression**  
  Earn XP based on workout intensity and calories burned  

- ⭐ **XP System**  
  Gain experience points from both study and fitness activities  

- 🆙 **Leveling System**  
  Level up as you accumulate XP  

- 🏆 **Achievements**  
  Unlock milestones for consistency, effort, and performance  

- 📊 **Stat Progression**  
  Build your personal stats like:  
  - Intelligence (📚 study)  
  - Strength (💪 workouts)  
  - Endurance (🔥 consistency)  

- 🎯 **Gamified Motivation**  
  Turn discipline into a rewarding game loop  

---

## 🛠️ Tech Stack

- Node.js – Backend logic  
- Electron – Desktop application  
- HTML / CSS / JavaScript – Frontend UI
